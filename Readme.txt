🏆 Трекер Привычек - React Native App
Мощное приложение для отслеживания привычек с продвинутой аналитикой, системой достижений и современным дизайном.
✨ Основные возможности
🎯 Управление привычками

✅ Создание и редактирование привычек с детальными настройками
📊 Отслеживание выполнения с календарем и статистикой
🔥 Система серий (streak) для мотивации
📱 Swipe-жесты для быстрого взаимодействия
🏷️ Категоризация и фильтрация привычек

📈 Аналитика и статистика

📊 Продвинутые графики и диаграммы
🗓️ Календарь с тепловой картой активности
📈 Анализ трендов и паттерн
